#  ImpasseWord - Secure Password Generator

A professional, client-side password generator that creates strong, random passwords instantly. Built with security and privacy as top priorities.

#  Features

- Customizable Passwords: Length 8-32 characters
- Character Selection: Uppercase, lowercase, numbers, symbols
- One-Click Copy: Secure clipboard integration
- Mobile Responsive: Works on all devices
- Zero Storage: No passwords stored or logged
- HTTPS Secure: Encrypted connections

#  Technologies Used

- Frontend: HTML5, CSS3, JavaScript
- Hosting: Netlify
- Security: Client-side generation, HTTPS
- Compatibility: Modern browsers with Clipboard API support
